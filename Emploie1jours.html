1ère semaine du stage :

1er jour :

Le matin :
Les personnes qui s'occupent de nous se sont présentées, nous avons fait un jeu pour "détendre l'atmosphère" et pour mieux nous connaître. Ensuite, nous avons visité l'école et installé le reste des applications nécessaires pour le stage.

L'après-midi :
Nous avons commencé notre portfolio et créé notre compte Github (sans aucun souci...).

2ème jour :

Le matin :
Nous avons continué notre portfolio puis nous avons eu une conférence sur la cybersécurité avec un faux site internet.

L'après-midi :
Nous avons fait un wordlguess (pareil que geoguesser) pour nous montrer qu'avec une simple image, on peut recueillir de nombreuses informations. Puis après le wordlguess, nous avons fait des énigmes qui utilisaient de nombreuses choses que l'on pouvait retrouver sur internet (déchiffrer des messages codés, retrouver une marque avec un bout d'image...).

3ème jour :

Le matin :
Nous avons fait du "poké poireau" (un jeu où le but est de coder l'IA de notre poireau pour le rendre plus fort). Puis nous avons eu une conférence sur l'IA et ses conséquences d'aujourd'hui (plus on utilise l'IA, plus on s'abrutit / L'IA transforme, elle ne remplace pas...).

L'après-midi :
Nous avons créé un code Python pour déchiffrer un code César puis nous avons utilisé Python Turtle pour dessiner notre "maison de stage" (champignon pour notre groupe).

4ème jour :

Le matin :
Nous avons utilisé des LEGO pour créer des canards pour nous montrer qu'avec seulement quelques briques, il y a des milliers de possibilités. Puis nous avons eu une conférence sur l'apprentissage (apprendre à apprendre).

L'après-midi :
Nous avons utilisé Arduino où le but était de recréer le jeu connu "Simon" avec des LED.

5ème jour :

Le matin :
Aujourd'hui, nous étions en distanciel, nous avons eu une conférence interactive sur l'IA (les erreurs qu'elle peut faire, ce n'est qu'un outil, pas un cerveau...).

L'après-midi :
Nous avons continué notre portfolio avec l'aide d'élèves.

Fin de la 1ère semaine.
